# Nix

See: [https://nixos.org](https://nixos.org)

## Installing Nix on MacOS

Installed via:

```sh
$ sh <(curl -L https://nixos.org/nix/install) --darwin-use-unencrypted-nix-store-volume --daemon
```

for a multi-user installation that works on Catalina.
